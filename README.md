# scmessages

Make the LaTeX2e messages (errors and warnings) cute.

## System Requirements

* TeX format: LaTeX2e
* expl3 environment

## Usage

Require package scmessages by `\usepackage`. That's all.

```tex
\usepackage{scmessages}
```
